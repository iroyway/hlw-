git常用操作

git常用操作

关联远程仓库

git remote add origin https://github.com/iroyway/hlw.git

取消关联远程仓库

git remote remove origin

git branch -M master

git push -u origin master

查看库中的所有分支

git branch -v

建立分支并进入分支

git checkout -b “分支名”
